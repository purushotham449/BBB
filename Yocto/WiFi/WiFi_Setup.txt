Step 1: Bring interface UP
    ip link set wlan0 up

Step 2: Kill existing wpa_supplicant
    killall wpa_supplicant
    rm -rf /var/run/wpa_supplicant

Step 3: Fix config file

    Edit:

        vi /etc/wpa_supplicant.conf
        Minimal working config:
        ctrl_interface=/var/run/wpa_supplicant
        update_config=1

        network={
            ssid="YOUR_WIFI_NAME"
            psk="YOUR_PASSWORD"
        }

Step 4: Start wpa_supplicant properly
    wpa_supplicant -B -i wlan0 -c /etc/wpa_supplicant.conf

Step 5: Verify connection
    iw dev wlan0 link

    Expected:

        Connected to <SSID> ✅

Step 6: Get IP
    udhcpc -i wlan0

Step 7: Verify IP
    ip addr show wlan0

Quick validation (just to confirm internet)

    Run:

        ping -c 3 8.8.8.8
        ping -c 3 google.com
